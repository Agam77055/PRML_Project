Chatbot using Classical Machine Learning

This project presents a rule-based and classical ML-driven chatbot for college-related FAQs. We built this system from scratch using custom tokenization, TF-IDF vectorization, and implemented models including Naïve Bayes, SVM, Decision Tree, Random Forest, and Cosine Similarity. The chatbot performs intent classification, entity extraction, and uses a dialogue manager to deliver contextual responses.
