ABSTRACT

ChatSphere is an advanced, modular chatbot framework developed entirely from scratch using Python, designed to assist college students‚Äîespecially first-years‚Äîin quickly accessing accurate institutional information. Leveraging classical machine learning models such as Multinomial Naive Bayes, Decision Trees, and Random Forests, ChatSphere offers robust intent classification, entity extraction, and context-aware response generation. With a custom-built dataset tailored to IIT Jodhpur's environment and deployment on cloud platforms using Docker, Flask, and Vercel, the system ensures scalability and ease of use. This project showcases the practical application of NLP and ML techniques in solving real-world information retrieval challenges in academic settings.

METHODOLOGY

We began by curating a custom dataset specific to institutional queries, structured with intents, sample patterns, and responses. Text data was preprocessed using a custom tokenizer that handled contractions, multi-word expressions, and punctuation. Feature extraction was performed using a TF-IDF vectorizer built from scratch. We implemented and evaluated three classifiers‚ÄîMultinomial Naive Bayes, Decision Tree, and Random Forest‚Äîfor intent classification. The backend was containerized using Docker and deployed on Google Cloud with Flask, while the frontend was built with Node.js and hosted on Vercel. Training was conducted using Colab Enterprise and Vertex AI with GPU acceleration.

DATASET

Our dataset is custom-built specifically for our institutional environment. Each entry is organized into an intent tag, a list of user input patterns, and a corresponding set of responses. This structure helps the chatbot map user queries to predefined intents with high accuracy. Preprocessing techniques such as synonym replacement and lexical augmentation enhance generalization. The dataset is modular and scalable, allowing easy updates and extensions.
